## 🛠️ Вариант 2: Самостоятельный хостинг OpenClaw

Для разработчиков, которые хотят полный контроль над инфраструктурой.

### Быстрая установка (Linux/macOS/Windows)

```bash
# macOS / Linux / WSL2
curl -fsSL https://openclaw.ai/install.sh | bash

# Windows (PowerShell)
iwr -useb https://openclaw.ai/install.ps1 | iex
```

### Системные требования

- **Node.js 24** (рекомендуется) или Node 22.14+
- **ОС:** macOS, Linux или Windows (нативно или WSL2)
- **ОЗУ:** Минимум 1 ГБ (рекомендуется 2 ГБ+)
